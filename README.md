# svm-enfermedad-cardiaca

# Predicción de Enfermedad Cardíaca con SVM

Este repositorio contiene un análisis de clasificación binaria para la predicción de enfermedad cardíaca utilizando Support Vector Machines (SVM) con kernel RBF. El foco del trabajo es un flujo de machine learning que incluye control de calidad de datos, preprocesamiento mediante pipelines, ajuste de hiperparámetros con validación cruzada y evaluación en un conjunto de prueba independiente.

El modelo se evalúa mediante ROC AUC, PR AUC y accuracy, e incluye un análisis de optimización del umbral (índice de Youden) y calibración de probabilidades (Brier score).
